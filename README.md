# CS 499 Computer Science Capstone

This project is the culmination of my what I have learned throughout the journey of pursuing my degree at SNHU. I was required to make a code review of a project I did earlier in the college program. I then had to make 3 enhancements to it in areas of software design, algorithms and data structures, and databases. This project consists of 4 milestones, and each milestone has a accompanying narrative file to explain the improvements made, as well as the lessons learned.
