# CS 499 Computer Science Capstone

This project is the culmination of my what I have learned throughout the journey of pursuing my degree at SNHU. I was required to make a code review of a project I did earlier in the college program. That project was a 3D object rendering program where the user could navigate a 3D space and see rendered 3D objects. I had to make 3 enhancements to it in areas of software design, algorithms, and databases. This project consists of 4 milestones. The first one is a code review video where I plan each enhancement, and the other 3 milestones are the enhancements themselves. Each enhancement has an accompanying narrative file to explain the improvements made, as well as the lessons learned.

## Milestone 1 - Code Review
[Code Review Video](https://www.youtube.com/watch?v=UfazH3vIsyE)

This video shows what plans I was making before I built each enhancement. I reviewed the code I had written for CS 330 and made some critiques as well as plans for improvement.


## Milestone 2 - Software Design & Engineering
[Milestone 2 Files](/Milestone2)

This enhancement focuses on improving the software design and re-engineering the code base. The code was refactored into C# from C++. It utilizes the OpenTK library for OpenGL functionality.

## Milestone 3 - Algorithms and Data Structures
[Milestone 3 Files](/Milestone3)

This enhancement adds an algorithm to generate a voxel data structure for 3D rendering. The algorithm generates a height map and produces voxels based on that map. Each voxel has a different color based on height to simulate a terrain.

## Milestone 4 - Databases
[Milestone 4 Files](/Milestone4)

This enhancement adds a database to store voxel maps in. A MongoDB database is used to store height map data. The user can select a slot, 1-9, and store a terrain in it.
